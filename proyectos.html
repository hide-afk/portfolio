Proyectos - 1º SMIR

Durante el primer año del grado de sistemas microinformáticos y redes, he realizado 5 proyectos distintos, los quales incluyen una gran variedad de aptitudes:

Proyecto 2 - Instalación y montaje de una oficina

He realizado el montaje y mantenimiento de un equipo de sobremesa y de un cableado estructurado, configurando la red local entre dos equipos.

Proyecto 4 - Web de gestión de incidencias

Simulando un taller de reparaciones, he hecho una página web con estética profesional mediante HTML y CSS, alojando esta misma en un servidor previamente preparado.



En esta página web se realiza una conexión a una base de datos mediante php, una gestión de tickets, y protocolos de resolución de averías.

Proyecto 5 - Sistema cloud de empresa de videojuegos.

He instalado y configurado un router para comunicar dos redes, simulando la red de una empresa de videojuegos.


Una red contenía un servidor TrueNAS, para hacer de sistema Cloud para los empleados de la otra red, conectados mediante un punto de acceso configurado.



Como tareas complementarias he realizado videojuegos 2D en Unity (programados con C#), una aplicación móvil con AppInventor.
